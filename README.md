# Project Charlie

Engineering school project : OpenGL / C++ / 3D

## Wanna test it ?

#### Make

	cd iSeason/build
	cmake ../project
	make

#### Run

	./template/game

##Prior installation

### CMake

    $ sudo apt-get install cmake


### Libraries Needed

####GLEW

    $ sudo apt-get install glew


####SDL2

    $ sudo apt-get install libsdl2-dev

####SDL2 MIXER

    $ sudo apt-get install libsdl2-mixer-dev
